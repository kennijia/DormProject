# DormProject

#### 介绍
基于Spring、SpringMVC、MyBatis三大Java框架开发的大学宿舍管理系统
#### 项目技术
前后端分离，后端遵循MVC的开发模式；
除了上述框架，还包含了JWT实现的用户认证和用户权限鉴别功能；
使用POI读取上传的Excel表格并将信息存储进数据库；
全局统一异常处理，全局统一响应结果。
